What is the difference between var, let, and const?

In JavaScript, var is function-scoped and can be re-declared; let and const are block-scoped, with let allowing re-assignment and const preventing it. However, const objects can have their contents modified.

What is the difference between parameters and arguments in functions

A parameter is a variable declared in a function's definition, while An argument is the actual value passed to a function when it's called.

What is the difference between global and local scope?

Global variables are shared among all functions and scripts, while Local variables are only accessible within the function or block where they're declared.
